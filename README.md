# dotfiles

## Prerequisites
- [Git](https://git-scm.com/)
- [GNU Stow](https://www.gnu.org/software/stow/)

## Install

```bash
$ cd $HOME
$ 
$ git clone https://github.com/wangyoucao577/dotfiles.git .dotfiles
$ cd .dotfiles
$ stow -v .
$ # Enjoy! 
```
